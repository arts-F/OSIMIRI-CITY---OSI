Quick Start

1️⃣ Local Development

# Install dependencies pnpm install # Start development server pnpm dev # Build for production pnpm build

2️⃣ Docker Setup

# Build image docker build -t osimiri-city:latest . # Run container docker-compose up -d # Access at http://localhost:3000

3️⃣ Deploy to Vercel

# Install Vercel CLI npm i -g vercel # Deploy to production vercel --prod
